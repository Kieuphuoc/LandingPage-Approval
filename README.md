# Landing Page Ký Số Trực Tuyến

## 📋 Giới thiệu

Landing page cho dịch vụ **Ký số trực tuyến (Digital Signature)** - Giải pháp ký kết hợp đồng điện tử hợp pháp, an toàn và nhanh chóng.

![SignTrust Banner](https://via.placeholder.com/1200x400/1e40af/ffffff?text=SignTrust+-+Ký+Số+Trực+Tuyến)

## ✨ Tính năng nổi bật

### 🎨 Thiết kế
- **Theme màu xanh** - Tạo cảm giác tin cậy, chuyên nghiệp
- **Light mode** - Giao diện sáng, dễ đọc
- **Modern SaaS style** - Phong cách hiện đại, cao cấp
- **Responsive** - Tương thích mobile, tablet, desktop

### 🎬 Animations
- Fade-in, slide-up khi scroll
- Hover effects cho buttons và cards
- Animated phone mockup với chữ ký động
- Step-by-step animation ở "Cách hoạt động"
- Glassmorphism effects
- Counter animation cho số liệu thống kê

## 🚀 Hướng dẫn sử dụng

### Cách 1: Mở trực tiếp
Mở file `index.html` bằng trình duyệt

### Cách 2: Live Server (VS Code)
1. Cài extension **Live Server**
2. Click chuột phải vào `index.html`
3. Chọn **Open with Live Server**

## 🔧 Tùy chỉnh

### Đổi màu chủ đạo
Chỉnh sửa CSS variables trong `style.css`:

```css
:root {
    --primary-500: #3b82f6;
    --primary-600: #2563eb;
    --primary-700: #1d4ed8;
    /* ... */
}
```

### Thêm logo công ty
Thay thế logo trong HTML:

```html
<a href="#" class="logo">
    <img src="your-logo.png" alt="Logo">
    <span class="logo-text">Tên công ty</span>
</a>
```
